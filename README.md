## Ritmo

Ritmo è un gestore di biblioteche personali che organizza i libri in base ai loro contenuti, oltre che in base alla loro veste editoriale.

La maggior parte dei software di gestione bibliografica tratta il libro come oggetto fisico — una edizione, un ISBN, un file. Ritmo parte dal contenuto: l'opera, gli autori, i temi, le relazioni tra testi. Una stessa opera in edizioni diverse è un solo contenuto. Un'antologia è un contenitore di contenuti distinti.

Ritmo è software libero, distribuito sotto licenza MIT. È scritto in Rust. È sviluppato per uso personale, senza compromessi di design per inseguire consenso.
